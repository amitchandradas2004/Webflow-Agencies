# 🌙 Cool Dark UI Landing Page(Static Page)

A modern, responsive, and clean dark-themed landing page template designed for Webflow agencies and creative digital brands.

## 🚀 Overview

This project is a sleek frontend landing page featuring:

- Bold Hero Section
- Call-To-Action Buttons
- Live Preview Section
- Product Statistics Showcase
- Testimonials
- Secondary CTA Banner
- Contact Form
- Footer Navigation

It is built with clean structure and modern UI principles to deliver a professional web presence.
